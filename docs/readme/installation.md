# インストール / アンインストール

## 前提

- Python 3.12 以上
- [uv](https://docs.astral.sh/uv/)

## インストール

[uv](https://docs.astral.sh/uv/) の tool インストールで、グローバルコマンド `todo` として導入します。

```sh
uv tool install .
```

開発リポジトリ内で動作確認だけしたい場合は、インストールせずに実行することもできます。

```sh
uv run todo --help
```

インストール後はどのディレクトリからでも `todo` で起動でき、常に同じデータベースを参照します。

## データの保存場所（重要）

> [!IMPORTANT]
> **データはあなたのホームディレクトリ配下に保存されます。**

「どのパスから実行しても同じデータにアクセスできる」を実現するため、実行ディレクトリではなく
**固定パス**に SQLite データベースファイル（`todo.db`）を置きます。場所は次の優先順位で決まります。

1. 環境変数 `MY_TODO_DB` が設定されていれば、そのパス（テスト・上書き用）
2. `$XDG_DATA_HOME/my-todo/todo.db`（`XDG_DATA_HOME` が設定されている場合）
3. `~/.local/share/my-todo/todo.db`（既定）

ディレクトリとデータベースは初回実行時に自動で作成されます。

## アンインストール

コマンド本体と、ホーム配下に残るデータの両方を消す必要があります。

```sh
# 1. コマンド本体を削除
uv tool uninstall my-todo

# 2. データを手動で削除（自動では消えません）
rm -rf ~/.local/share/my-todo
```

> [!NOTE]
> `uv tool uninstall` で消えるのはコマンド本体だけです。
> **データベースを置いたディレクトリは自動では削除されない**ため、
> データごと消したい場合は上記のように手動で削除してください。
> `XDG_DATA_HOME` や `MY_TODO_DB` を設定して場所を変えていた場合は、そのパスを消してください。
