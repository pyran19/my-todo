# my-todo

コマンドラインで使う、**自分の欲しい機能だけ**を備えた最小TODO。

世の中にTODOアプリは数えきれないほどありますが、「過剰な機能はかえって認知負荷」という考えのもと、
必要な部分だけを自作したものです。ターミナルから `todo` で起動でき、どのディレクトリから実行しても
同じデータを参照します。

## 特徴

- **どこからでも同じデータ。** どのパスで実行しても同一のストレージにアクセス。
- **日付は自動記録。** 作成日・更新日は自動。ユーザーは意識しない。
- **おおまかな寿命で分類。** 古くなったタスクは短期→中期→長期へ自動で移行。締切のきっちり管理はしない。
- **`$EDITOR`（既定 vim）で編集。** 長い本文もエディタで書ける。
- **to think リスト。** すぐ片付かない「考えごと」は通常タスクと別立てで管理。

## クイックスタート

```sh
# uv の tool インストールでグローバルコマンド化
uv tool install .

# タスクを追加
todo add "牛乳を買う"

# 一覧（引数なしでも一覧）
todo
```

> [!IMPORTANT]
> **データはホームディレクトリ配下（既定 `~/.local/share/my-todo/`）に保存されます。**
> アンインストール時はコマンド本体を消すだけではデータが残るため、手動での削除が必要です。
> 詳しくは [インストール / アンインストール](./docs/readme/installation.md) を参照してください。
